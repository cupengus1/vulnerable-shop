# Hướng Dẫn Thêm Hình Ảnh Sản Phẩm

## Lưu ý
Folder này cần chứa hình ảnh sản phẩm. Hiện tại database sử dụng các tên file sau:

- `ao-thun-nam-1.jpg`
- `ao-somi-nam-1.jpg`
- `quan-jeans-nam-1.jpg`
- `ao-thun-nu-1.jpg`
- `vay-lien-nu-1.jpg`
- `quan-jean-nu-1.jpg`
- `ao-khoac-nam-1.jpg`
- `dam-maxi-nu-1.jpg`
- `ao-polo-nam-1.jpg`
- `chan-vay-nu-1.jpg`
- `placeholder.jpg` (ảnh mặc định)

## Cách thêm ảnh

### Option 1: Tự tìm ảnh
1. Tìm ảnh sản phẩm từ nguồn miễn phí (unsplash.com, pexels.com)
2. Đổi tên theo danh sách trên
3. Copy vào folder này

### Option 2: Sử dụng placeholder
1. Tạo ảnh 500x500px với background màu nhạt
2. Thêm text tên sản phẩm
3. Lưu với tên tương ứng

### Option 3: Để mặc định
- Website sẽ tự động sử dụng `placeholder.jpg` nếu không tìm thấy ảnh
- Chỉ cần tạo 1 file `placeholder.jpg` 500x500px là đủ

## Khuyến nghị kích thước
- **Width**: 500-800px
- **Height**: 500-800px
- **Format**: JPG hoặc PNG
- **Size**: < 200KB mỗi ảnh
